# evaluate-password

> A lightweight module for evaluating passwords

## Installation

```
npm install evaluate-password
```

## Usage

```js
var evaluatePassword = require('evaluate-password');

evaluatePassword('j5JSxDUBBkYz!D1$I&'); // 39
```
